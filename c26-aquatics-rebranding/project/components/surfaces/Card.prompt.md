Rounded content card with an optional media area on top. Use `interactive` for clickable program/age tiles.

```jsx
<Card media="/coach.jpg" interactive>
  <h3>Youth Development</h3>
  <p>Stroke technique for ages 6–12.</p>
</Card>
```

`media` accepts an image URL or any node. `tone="dark"` for navy cards on light sections.
