Athletic, uppercase call-to-action button — squared corners, condensed type. Use `primary` (orange) for the hero action; reserve one per view.

```jsx
<Button variant="primary" size="lg">Book a session</Button>
<Button variant="outline">Explore programs</Button>
```

Variants: `primary` (orange + glow), `secondary` (teal), `dark` (navy), `outline`, `outline-light` (for dark backgrounds), `ghost`. Sizes: `sm` / `md` / `lg`. Pass `full` to stretch, `as="a"` + `href` for link CTAs.
