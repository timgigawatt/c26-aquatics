# Swim Lessons UI kit — Option 1 (SEO depth)

Hi-fi recreation of the approved **Option 1** Swim Lessons direction — built to rank and to educate.

## Files
- `index.html` — mounts the interactive page (the `@dsCard` preview).
- `SwimLessonsView.jsx` — full composed page, exported as `SwimLessonsView`. Sections: `TopBar`, `SubNav` (breadcrumb + in-page tabs), `Hero`, `Benefits`, `Philosophy`, `AgeGroups` (link to child pages), `Pricing`, `FAQ`, `FinalCTA`.

## Composes
`Button`, `Badge`, `Card` from the design system.

## Interactions
- **Sub-nav tabs** (Overview / Ages / Pricing / FAQ) highlight on click.
- **FAQ** accordion expands one answer at a time.
- **Age-group cards** are interactive and point at child URLs (`/swim-lessons/infant`, etc).

## Placeholders
Photography is hatched placeholder blocks. FAQ answer copy and exact pricing are stubbed for you to fill.
